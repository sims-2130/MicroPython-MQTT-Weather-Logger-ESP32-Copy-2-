# MicroPython-MQTT-Weather-Logger-ESP32-Copy-2-
MicroPython MQTT Weather Logger (ESP32) – A Wokwi‑based IoT project that logs temperature and humidity from a DHT sensor on an ESP32 and publishes the data to an MQTT broker. Demonstrates MicroPython, Wi‑Fi connectivity, sensor reading, and lightweight MQTT messaging for remote weather monitoring.
